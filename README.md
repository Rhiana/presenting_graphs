# README

This is a demo website for collecting various SVG graphs built with ruby

* Ruby version
  * 2.5.1
* Rails
  * 5.2.0

* Set up
```
git clone git@github.com:Rhiana/presenting_graphs.git
cd presenting_graphs
bundle install
rake db:migrate
rails s
```

* Testing
```
 rake spec
```

* Deployment instructions
```
 git push heroku master:master
```
